# Stickers Directory

هذا المجلد يحتوي على ملفات الـ animated emoji (.tgs files).

## كيفية إضافة Stickers

### الطريقة 1: عبر البوت (للمسؤولين)

1. أرسل `/upload_sticker` للبوت
2. أرسل ملف `.tgs` مع إضافة اسم الـ sticker في الـ caption
3. الأسماء المتاحة:
   - `plans` - للخطط والأسعار
   - `subscribe` - للاشتراك
   - `subscriptions` - للاشتراكات
   - `referral` - لنظام الإحالة
   - `info` - للمعلومات
   - `back` - لزر الرجوع
   - `payment` - للدفع
   - `trial` - للتجربة المجانية
   - `confirm` - لتأكيد الدفع
   - `success` - للنجاح
   - `error` - للأخطاء
   - `warning` - للتحذيرات
   - `trc20` - لشبكة TRC20
   - `bsc` - لشبكة BSC

### الطريقة 2: يدوياً

1. ضع ملفات `.tgs` في هذا المجلد
2. استخدم الأسماء التالية:
   - `plans.tgs`
   - `subscribe.tgs`
   - `subscriptions.tgs`
   - `referral.tgs`
   - `info.tgs`
   - `back.tgs`
   - `payment.tgs`
   - `trial.tgs`
   - `confirm.tgs`
   - `success.tgs`
   - `error.tgs`
   - `warning.tgs`
   - `trc20.tgs`
   - `bsc.tgs`

## ملاحظات

- ملفات `.tgs` هي تنسيق Telegram للـ animated stickers
- يمكنك تحويل ملفات Lottie (.json) إلى .tgs باستخدام أدوات مثل `lottie2tg`
- البوت سيرسل الـ stickers تلقائياً عند توفرها
- إذا لم يكن الـ sticker متوفر، سيتم إرسال الرسالة بدون sticker

## أدوات مفيدة

- [lottie2tg](https://github.com/ed-asriyan/lottie2tg) - لتحويل Lottie إلى .tgs
- [TGS to GIF](https://tgs-to-gif.com/) - لمعاينة الـ stickers
